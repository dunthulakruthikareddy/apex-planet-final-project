# ShopMart - Responsive E-Commerce Website

## 📌 Project Overview
ShopMart is a responsive e-commerce website developed as the Final Capstone Project for the ApexPlanet Web Development Internship. The project demonstrates modern front-end web development skills using HTML, CSS, and JavaScript.

## ✨ Features
- Responsive Design
- Attractive Home Page
- Navigation Bar
- Featured Products Section
- Add to Cart Functionality
- Mobile-Friendly Layout
- Performance Optimized
- Cross-Browser Compatible

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 🎯 Learning Outcomes
- Responsive Web Design
- HTML5 Page Structure
- CSS Styling and Flexbox
- JavaScript Functions and Events
- Performance Optimization
- Cross-Browser Compatibility
